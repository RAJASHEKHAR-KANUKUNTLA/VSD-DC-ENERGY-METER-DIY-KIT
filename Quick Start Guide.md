# VSD DC Energy Meter DIY Kit

The **VSD DC Energy Meter DIY Kit** is an Educational hardware kit designed to measure and visualize DC electrical parameters such as **Voltage, Current, Power, and Energy Consumption**.

The system uses the **VSD Ultra Board** along with sensors and a 16-bit ADC module to capture accurate electrical readings. The measured data is displayed on an **LCD screen** and also visualized on a **Processing-based graphical dashboard** in real time.

---

# Quick Start Guide

## 1. Unpack the Kit

Carefully unpack the **VSD DC Energy Meter DIY Kit** and verify that all components are present.

### Components Included

1. PCB Panel  
2. **VSD Ultra Board**  
3. 16-bit ADC Module  
4. Voltage Sensor  
5. Current Sensor  
6. LCD Display  
7. Connecting Wires  
8. 12V Adapter  
9. DC Motor (Load)  
10. Additional Connecting Wires  

---

## 2. Assemble the Hardware

Place the components on the PCB panel and connect them according to the provided circuit diagram.

### Important Connections

- Connect the **Voltage Sensor in parallel** with the DC supply.
- Connect the **Current Sensor in series** with the load.
- Connect the **ADC module** to the sensors.
- Connect the **ADC module to the VSD Ultra Board**.
- Connect the **LCD display** to the VSD Ultra Board.

Make sure all connections are secure before powering the system.

---

## 3. Power the System

- Connect the **VSD Ultra Board** to your computer using a **USB cable**.
- Provide **12V DC supply** to:
  - Voltage sensor
  - Current sensor
  - Load (DC Motor)

---

## 4. Upload the Firmware

1. Open **Arduino IDE** on your computer.
2. Connect the **VSD Ultra Board** via USB.
3. Select the correct **board and serial port**.
4. Upload the **DC Energy Meter firmware**.

---

## 5. Start the Processing Dashboard

1. Open the **Processing software**.
2. Run the **VSD DC Energy Meter dashboard program**.
3. Select the correct **serial port** used by the VSD Ultra Board.

---

## 6. Begin Monitoring

Once the system starts running, the kit will measure and display:

- **Voltage (V)**
- **Current (I)**
- **Power (P)**
- **Energy Consumption (Wh)**

These values will appear on:

- LCD Display  
- Processing Graphical Dashboard  

---

## 7. Observe Real-Time Data

The Processing interface provides:

- Live readings
- Real-time graphs
- Energy consumption visualization

This allows users to analyze the electrical behavior of the connected **DC load (motor)**.

---

## 8. Shut Down the System

After completing the experiment:

1. Turn off the **12V power supply**.
2. Disconnect the **USB cable**.
3. Safely disconnect all components.
